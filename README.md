# GAT4-POC

A POC for the GAT-4 project — geolocated health data visualization.

## Prerequisites

- [Nix](https://nixos.org/download) with flakes enabled
- [nix-direnv](https://github.com/nix-community/nix-direnv) installed in your system Nix config (NixOS or home-manager)
- [direnv](https://direnv.net/) hooked into your shell

## Getting started

```bash
direnv allow
```

That's it. The dev environment will be built and activated automatically on every entry into the project directory.
